Kesimpulan: Jika radio button diberi atribut "name" yang sama, maka elemen tersebut menjadi satu grup dan pengguna hanya bisa memilih salah satu opsi (single-choice). Jika nama atributnya berbeda, pengguna bisa mencentang semuanya secara bersamaan.

Uji coba 3



Kesimpulan: Checkbox memungkinkan pengguna untuk memilih lebih dari satu pilihan sekaligus (multiple-choice), tidak peduli apakah atribut "name" diset sama atau berbeda. Namun, memberikan atribut "name" yang sama berguna untuk mengelompokkan data saat dikirim ke server.

Uji coba 4




Kesimpulan: type="color" memunculkan palet warna (color picker), type="date" memunculkan kalender interaktif untuk memilih tanggal, dan type="button" ketika ditekan akan menjalankan script (dalam hal ini memunculkan popup peringatan "Hello World!").

Uji coba 5




Kesimpulan: type="datetime-local" digunakan untuk memilih tanggal lengkap dengan jamnya. type="month" digunakan untuk memilih bulan dan tahun. type="email" akan memvalidasi input secara otomatis, dan akan memunculkan peringatan jika teks yang diinput tidak mengandung karakter '@' layaknya format email standar.

Uji coba 6




Kesimpulan: type="image" berfungsi seperti tombol submit namun divisualisasikan menggunakan gambar. type="file" digunakan untuk mengunggah (upload) dokumen dari komputer. type="hidden" digunakan untuk menyimpan dan mengirim data ke server secara diam-diam tanpa menampilkannya pada halaman web.

Uji coba 7



Kesimpulan: type="number" membatasi input hanya berupa angka sesuai batas min dan max. type="range" menampilkan input dalam bentuk slider geser. Jika nilai min diubah menjadi -50, maka batas minimal dari slider atau input angka bergeser turun dan bisa menerima nilai negatif hingga -50.

Uji coba 8




Kesimpulan: type="url" memastikan bahwa data yang diketik berupa tautan valid (harus mengandung http:// atau https://). type="search" adalah input teks biasa dengan tambahan tombol 'x' kecil untuk menghapus teks dengan cepat. type="tel" dikhususkan untuk nomor telepon, dan dengan bantuan atribut "pattern", input bisa divalidasi harus sesuai format tertentu (misal: xxx-xx-xxx).

Uji coba 9



Kesimpulan: type="time" menampilkan antarmuka khusus untuk memasukkan jam dan menit. Sementara type="week" memunculkan kalender di mana pengguna hanya bisa memilih spesifik urutan minggu ke-berapa di dalam tahun tertentu. Keduanya sangat memudahkan penyeragaman format waktu dan meminimalisir kesalahan pengetikan pengguna.
